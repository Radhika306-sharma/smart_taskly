# Smart Taskly

Smart Taskly is an AI-powered task management and calendar application built with React, TypeScript, and Google Gemini. It allows users to create, organize, and manage tasks through a modern calendar interface while also supporting natural language task creation using AI.

This project was built to explore how AI can simplify personal productivity by converting everyday language into structured tasks and schedules.

## Features

- Month, Week, and Day calendar views
- Create, edit, and delete tasks
- Recurring tasks support
- Drag-and-drop task rescheduling
- Task prioritization and categorization
- Search and filter functionality
- Dark mode interface
- AI assistant powered by Google Gemini
- Local storage persistence

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Google Gemini API
- Framer Motion
- date-fns
- Lucide React

## Getting Started

Clone the repository:

```bash
git clone git@github-vit:Radhika306-sharma/smart_taskly.git
cd smart_taskly
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
